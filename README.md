# Snitch Clothing Brand Website Clone

A full-stack clone of the Snitch clothing brand website.

## Project Structure

```
Snitch/
├── frontend/          # Angular application
│   └── snitch-app/   # Main Angular project
└── backend/          # Backend API (coming soon)
```

## Frontend

The frontend is built with **Angular** and includes:
- Routing enabled
- SCSS for styling
- Modern Angular architecture

### Getting Started with Frontend

```bash
cd frontend/snitch-app
npm start
```

The application will be available at `http://localhost:4200`

### Build for Production

```bash
cd frontend/snitch-app
npm run build
```

## Backend

The backend will be implemented in the future. Check the `backend` folder for updates.
